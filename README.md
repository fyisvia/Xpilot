# Xpilot

Xpilot是基于vue3+tailwindcss+daisyui开发的一个日本麻将（Riichi Mahjong）工具性网站，旨在帮助日麻玩家提升雀力。项目整合了牌效率分析等实用功能，并针对各个用户端的用户体验进行了优化和改进，提供更便捷、直观的日麻学习与实战支持。

## 主要功能

- **牌效率分析**：自动分析当前手牌，给出最优舍牌建议，提升和牌速度与效率。
- **点数计算练习**：自动随机生成番数及符数，帮助玩家训练实战中计算能力。
- **手牌点数计算练习**：自动随机生成手牌，根据完整手牌练习判定役、符与点数，帮助玩家训练实战中计算能力。
- **清一色听牌练习**：自动随机生成手牌，帮助玩家训练对于清一色牌型的判断力和实战应对能力。
- **牌效率练习**：练习各种情况下分析在牌效率角度下的最优解。
- **基本术语词典**：快速查阅常见日麻术语释义。。
